Mushroom Prediction using ML

This project predicts if a mushroom is poisonous or edible based on its physical description, such as cap shape and color. We used Pandas and Matplotlib for data cleaning and EDA, while scikit-learn handled OneHotEncoding and model training. The application was deployed using Flask to create an interactive user interface. The dataset was sourced from Kaggle to train the classification model.

Sentiment Analysis of Phone Reviews

This project analyzes Amazon/Flipkart reviews to evaluate product quality by categorizing customer feedback into positive and negative sentiments. Using Transformers (NLP) and Pandas, it calculates ratings on a 5-point scale and generates ranking tables for mobile phones based on feature categories. The application is deployed via Streamlit, providing an interactive UI for visualizing consumer sentiment data.
